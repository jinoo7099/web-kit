# Day-8

## Cafe

### 구현 사항

```

카페를 구현합니다.

카페는 손님, 가게, 바리스타로 나눠져 있습니다.

가게에는 음료, 디저트등 자유로운 메뉴가 있습니다.
음료는 기본 음료가 존재합니다

손님은 가게의 음료를 주문 가능합니다.
주문 후 주문번호를 받습니다.

주문이 접수되면 바리스타에게 할당 됩니다.
바리스타는 최소 1명 존재합니다.
바리스타는 모든 음료수를 각자 지정된 시간내에 만들 수 있습니다.
(모든 음료수를 만드는 시간은 동일합니다. 바리스타마다 시간이 다를 뿐 입니다.)

바리스타는 한번에 음료수를 1개만 만들 수 있습니다.
만들고 있는 음료수가 있다면 주문을 받을 수 없습니다.

상황따라 먼저 접수된 주문이 만들어지는 바리스타에 따라 받는 순서가 달라질 수 있습니다.

```

### 제한사항

- 클래스를 사용하지 않습니다.
- 디자인은 자유롭게 진행해 주세요.
- 다른 프레임워크 라이브러리 ( React, View, Angular ) 를 사용하지 않습니다.
- 손님, 주문, 바리스타는 DOM으로써 보여져야 합니다.
